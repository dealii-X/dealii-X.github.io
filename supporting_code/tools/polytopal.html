polytopal (Polydeal)

Polytopal discontinuous Galerkin building blocks with agglomeration and multigrid support, used to explore general mesh discretizations and reduced-complexity workflows.

Homepage Repository

  • Focus: polygonal and polyhedral DG discretizations built on top of deal.II.

  • Highlights: agglomeration-based multigrid, MPI support, R-tree driven mesh hierarchy generation.

  • Local path: tools/polygonal
